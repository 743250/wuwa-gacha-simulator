## 联网助手第四次批注：角色技能/共鸣特性核心玩法

> 用户想要的是游戏内“技能介绍”那种机制描述，不是完整倍率表。下面按模拟器可落地的粒度整理：核心循环、共鸣技能、共鸣解放、共鸣回路、变奏/延奏。注意：这是基于 Fandom/Prydwen 技能页提炼的中文摘要，不是官方中文原文逐字搬运。

### 建模总原则

- 每个角色先做 1 个核心资源或核心状态，不要把所有派生段都塞进回合制。
- 共鸣回路 = 角色独有玩法条，应优先模拟；倍率细节可以后补。
- 变奏/延奏可以简化为“入场伤害/出场给下个角色 Buff”，不必做完整动作链。
- 对拥有异常状态的角色，优先保留状态玩法：气动侵蚀、衍射失序、热熔爆发等。

### 10 个角色核心玩法摘要

#### 忌炎

- 核心循环：攒破阵值，开共鸣解放进入“青龙/破阵”输出窗口，用强化重击长枪连段打 AOE。
- 共鸣技能：向前突进造成气动伤害，可作为攒资源和补伤害手段。
- 共鸣解放：进入强化状态，普通输出轴变成重击连段；主要伤害按重击类型结算。
- 共鸣回路：破阵值达到条件后强化重击或技能，模拟器可做成 `resolve` 资源，满值后下次重击倍率提高。
- 延奏技能：给下一名角色重击伤害加成；适合接长离、今汐等需要爆发窗口的角色。
- 模拟器建议：定位主 C；核心标签 `aoe / heavy_attack / aero / resolve`。

#### 今汐

- 核心循环：队伍行动帮今汐积累韶光/Incandescence，今汐入场后打共鸣技能链，最后释放高额衍射爆发。
- 共鸣技能：多段技能链，进入乘岁凌霄/特殊空中状态后继续派生，核心伤害集中在最后的强化技能段。
- 共鸣解放：大范围衍射爆发，作为补充核爆。
- 共鸣回路：队友造成属性伤害时积累韶光；满层后强化核心技能。模拟器可做成 `incandescence` 层数，队友每次攻击 +1，满层今汐技能变强。
- 延奏技能：对下一名角色提供技能相关增益可简化为“下个角色共鸣技能伤害提高”。
- 模拟器建议：定位主 C；核心标签 `spectro / skill_nuke / team_charge`。

#### 长离

- 核心循环：通过普攻/技能进入心眼状态，积累离火，满离火后释放强化重击“焚身以火”，再用解放补层和爆发。
- 共鸣技能：突进/派生热熔伤害，并可进入心眼状态。
- 共鸣解放：热熔爆发，同时补充核心资源，让后续强化重击更顺。
- 共鸣回路：离火层数是核心；满层消耗释放强化重击。模拟器可做成 `enflamement` 4 层，满层后重击变成强力热熔爆发。
- 延奏技能：给下一名角色热熔伤害和共鸣解放伤害加成。
- 模拟器建议：定位主 C/副 C；核心标签 `fusion / skill / liberation_buff / enflamement`。

#### 守岸人

- 核心循环：用技能和解放维持治疗与领域，攒协奏后触发强化入场/团队暴击增益。
- 共鸣技能：治疗全队，并召唤星蝶追踪攻击。
- 共鸣解放：展开领域，提供持续治疗，并根据阶段给队伍暴击率/暴击伤害等增益。
- 共鸣回路：围绕星蝶、领域阶段、协奏触发强化入场。模拟器可简化为 `stellarealm` 领域状态，持续数回合，每回合治疗并给队伍暴击 Buff。
- 延奏技能：为队伍提供暴击相关增益，属于强辅助出口。
- 模拟器建议：定位治疗/辅助；核心标签 `heal / crit_buff / field / spectro`。

#### 椿

- 核心循环：切换形态并围绕普攻输出，积累花蕾/红椿资源后进入强化爆发窗口。
- 共鸣技能：切换攻击姿态或进入特殊移动/输出形态，改变普攻连段。
- 共鸣解放：湮灭范围爆发，通常用于输出窗口收尾。
- 共鸣回路：资源满后强化普攻或进入更高伤害状态。模拟器可做成 `bud` 或 `camellia` 层数，普攻叠层，满层获得“盛放”状态。
- 延奏技能：可简化为提高下一角色普攻或湮灭相关伤害。
- 模拟器建议：定位主 C；核心标签 `havoc / basic_attack / stance / burst_window`。

#### 珂莱塔

- 核心循环：用佩枪攻击和共鸣技能积累晶体/灵感类资源，在窗口期释放强化技能或高倍率枪击。
- 共鸣技能：冷凝伤害核心技能，适合作为主要爆发段。
- 共鸣解放：高额冷凝爆发，配合技能窗口使用。
- 共鸣回路：围绕晶体资源和强化技能。模拟器可做成 `crystal` 层数，技能命中叠层，满层下次技能额外伤害。
- 延奏技能：可简化为给下一角色共鸣技能伤害或冷凝相关增益。
- 模拟器建议：定位主 C；核心标签 `glacio / pistol / skill_damage / crystal`。

#### 菲比

- 核心循环：给敌人施加衍射失序/光噪类状态，再通过普攻、重击或技能放大该状态伤害。
- 共鸣技能：衍射伤害，并服务于状态施加或形态切换。
- 共鸣解放：衍射范围爆发，适合作为状态铺好后的结算技能。
- 共鸣回路：围绕衍射异常状态与输出姿态。模拟器可做成 `spectro_frazzle` 标记，菲比攻击标记目标时额外伤害。
- 延奏技能：可简化为放大目标身上的衍射失序伤害。
- 模拟器建议：定位主 C/异常输出；核心标签 `spectro / frazzle / rectifier / status_amp`。

#### 卡提希娅

- 核心循环：积累决意，在共鸣解放中化身芙露德莉斯，进入显化状态；显化期间用强化普攻/技能叠加并结算风蚀效果。
- 共鸣技能：气动伤害和牵引/位移，可服务于叠风蚀或聚怪。
- 共鸣解放：若决意满 120，进入芙露德莉斯形态；若未满则进入卡提希娅显化状态。生命值会被压到上限 50%，但低血时不再继续消耗生命。
- 共鸣回路：芙露德莉斯攻击积攒决意，满 120 后释放看潮怒风哮之刃。模拟器可做成 `resolve_120`，满值开大进入强化 2-3 回合。
- 特殊状态：显化期间部分攻击会结算并减少目标风蚀层数，保留“气动侵蚀/风蚀层”很重要。
- 模拟器建议：定位主 C；核心标签 `aero / erosion / transform / hp_cost / resolve`。

#### 嘉贝莉娜

- 核心循环：热熔佩枪主 C，围绕重击/枪击输出与强化窗口打爆发。
- 共鸣技能：热熔伤害，可作为进入输出循环或补资源手段。
- 共鸣解放：热熔爆发，适合在资源或 Buff 准备好后使用。
- 共鸣回路：资料仍需继续核验；模拟器可先做成 `hunt` 或 `heat` 资源，普攻/技能叠层，满层强化重击。
- 延奏技能：可简化为热熔伤害或重击相关增益。
- 模拟器建议：定位主 C；核心标签 `fusion / pistol / heavy_attack / heat`。标注 `[技能细节待查]`，不要写死官方术语。

#### 卡卡罗

- 核心循环：常态用普攻/技能积累残忍/杀意类资源，开共鸣解放进入杀戮武装状态，打高频导电连段和死告。
- 共鸣技能：多段导电攻击，可连续使用，作为常态主要输出和攒资源手段。
- 共鸣解放：进入强化状态，普攻链变成更强的导电连段，输出节奏明显改变。
- 共鸣回路：资源满后可施放强化重击/仁慈或死告类攻击。模拟器可做成 `killing_intent`，常态技能叠层，解放期消耗层数追加伤害。
- 延奏技能：可简化为下一角色导电伤害或共鸣技能伤害增益。
- 模拟器建议：定位主 C；核心标签 `electro / liberation_mode / combo / killing_intent`。

### 代码落地建议

可以给每个角色加一个轻量机制配置，不要一开始写复杂脚本：

```js
{
  forte: {
    key: 'resolve',
    max: 120,
    gain: { attack: 10, skill: 30 },
    spend: 'burstMode'
  },
  tags: ['aero', 'heavy_attack', 'transform'],
  loop: '攒资源 -> 开大/强化 -> 爆发窗口'
}
```

优先落地顺序：

1. 守岸人：领域治疗 + 暴击 Buff，最容易提升队伍体验。
2. 今汐：队友帮攒层 + 技能核爆，最像真实玩法。
3. 卡提希娅：决意 120 + 变身 + 风蚀，非常贴合截图。
4. 忌炎/长离/椿：主 C 输出窗口。
5. 珂莱塔/菲比/嘉贝莉娜/卡卡罗：后续按资源条补。
