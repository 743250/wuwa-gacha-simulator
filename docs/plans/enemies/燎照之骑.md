# 燎照之骑 移植计划

> 官方数据: [../../sources/enemies/world-boss-mechanics.md](../../sources/enemies/world-boss-mechanics.md)
> mechanic type: `inferno_marks`
> 复杂度: 中
> 元素: 热熔 | Lv90 HP: 468,488 | ATK: 8,912 | DEF: 1,512 | 热熔40%/他10%

## 官方核心

Fusion Burst 叠层(10 层满爆)+ 双阶段(骑摩托 → 步行 @75%HP)

## 模拟器抽象

灼伤标记(Fusion Burst 折合):
- 被热熔攻击命中 +N 层灼伤(叠层本身不扣血)
- 每层灼伤:回合结束时扣血 = 攻击×80%(dot)
- 上限 5 层
- 叠满 5 层:立即清空层数,额外爆炸 = 攻击×300%

双阶段:
- 阶段一(HP ≥ 75% · 骑摩托):每次攻击附带 2 层灼伤
- 阶段二(HP < 75% · 步行):每次攻击给全体角色各 1 层灼伤(群体叠层压力)

## 参数

```js
mechanic: {
  type: 'inferno_marks',
  markDmgPct: 0.80,
  maxStacks: 5,
  burstMult: 3.0,
  phase1Stacks: 2,
  phase2Stacks: 1,
  threshold: 0.75
}
```
