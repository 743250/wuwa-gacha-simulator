# 梦魇亚当·重锤 移植计划

> 官方数据: [../../sources/enemies/world-boss-mechanics.md](../../sources/enemies/world-boss-mechanics.md)
> mechanic type: `adam_splash_defdown`
> 复杂度: 低
> 元素: 物理 | Lv90 HP: 500,000 | ATK: 11,000 | DEF: 1,512 | 物理40%/他10%

## 官方核心

重锤溅射 + 降防 + 低血狂暴

## 模拟器抽象

溅射伤害:
- 所有攻击对主目标外相邻角色造成 50% 溅射伤害
- 3 人队:打位置 1 → 位置 2 受溅射

降防叠加:
- 每次被命中:DEF -10%(叠乘,最多 3 层)
- 持续 2 回合(可刷新)
- 切换角色后新角色 DEF 正常(debuff 挂角色不挂队)

低血狂暴:
- HP < 40%:ATK +50%,每回合攻击 2 次(双动)
