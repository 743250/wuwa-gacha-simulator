# 哀声鸷 移植计划

> 官方数据: [../../sources/enemies/world-boss-mechanics.md](../../sources/enemies/world-boss-mechanics.md)
> mechanic type: `aix_dive`
> 复杂度: 中
> 元素: 衍射 | Lv90 HP: 435,137 | ATK: 10,386 | DEF: 1,512 | 衍射40%/他10%

## 官方核心

追踪弹 ×3 + 飞行俯冲(可弹反 → 瘫痪)

## 模拟器抽象

追踪弹 ×3:
- 每 3 回合,发射 3 枚追踪弹
- 随机选择 3 个目标(可重复),每发 = ATK × 70%

飞行俯冲(特色弹反窗口):
- 每 5 回合,飞起俯冲当前角色
- 伤害 = ATK × 200%
- 若本回合角色使用重击 → 弹反成功
- 弹反成功 → BOSS 瘫痪 1 回合(跳过下回合攻击 + 受到伤害 +50%)
