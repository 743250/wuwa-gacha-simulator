# 聚械机偶 移植计划

> 复杂度: 低

## 官方核心

残骸眩晕(捡碎片砸晕)+ 风壁减伤 + "最弱 Overlord"

## 模拟器抽象

残骸掉落(Unique 机制):
- 每 5 回合,BOSS 身上掉下金属残骸
- 下回合玩家可选择"投掷残骸"(替代普攻,0 AP)
- 投掷残骸 → BOSS 眩晕 1 回合(跳过攻击 + 受伤 +50%)
- 这是"免费输出窗口",体现"最简单 Overlord"

风壁:
- 每 4 回合,召唤风壁减伤 40%,持续 2 回合

旋转 AOE:
- 每 3 回合,大型回旋 = ATK × 90%(全队)
- 后摇长:下回合 BOSS 攻击 -30%
