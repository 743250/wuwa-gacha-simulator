# 桃祈 移植计划

> 状态: ✅ 已实装
> 官方技能、共鸣链名称与面板以 `docs/sources/characters/individual/桃祈.json` 为准
> 强度 Tier: **C**
> 移植级别: A → 工厂（customLines + chainEffects · C-Tier 数值最小化）
> 定位: 湮灭 · 长刃 · 辅助 · **「磐岩护壁 · 攻防转换反击」**

## 模拟器核心

桃祈是 4★ 坦克型辅助（C-Tier）。FORTE gauge「守势」max 100 → enhancedSkill ×1.6。技能展开「磐岩护壁」提供护盾（1 链 hp +12%），6 链护盾期间普攻/重击 +40%。重击触发「攻防转换」消耗守势造成伤害 + 回血 + 防御 +50%（4 链）。C-Tier 满链 ×3.5。循环：开护盾 → 叠守势 → 攻防转换反击。


## 模拟器保留

| 官方系统名 | 模拟器抽象 |
|---|---|
| **重器藏锋** | 普攻按湮灭伤害结算，护盾期间由 6 链提高普攻/重击伤害 |
| **固若金汤** | 共鸣技能展开护盾，作为护盾型辅助入口 |
| **攻防转换** | 重击发后制人抽象为 2 AP 重击反击 |
| **不动如山** | 共鸣解放按主目标/副目标伤害结算 |
| **千钧** | 延奏的技能伤害加深并入辅助定位，不单独做持续状态 |

## 模拟器舍弃/合并

| 官方机制 | 处理 |
|---|---|
| 护盾量、攻防转换精准触发窗口 | 合并为护盾入口与重击反击，避免回合制里维护受击判定窗口 |
| 防御倍率细分 | 合并进 C-Tier 的固定倍率与 `def` 标准 effect |
| 抗打断、受击减伤细节 | 不单独模拟，只保留护盾/防御方向 |
