## 模拟器核心

秋水是 4★ 副C（B-Tier），核心为召唤「雾化分身」嘲讽敌人 + 分身存在期间本体普攻穿过「虚实之门」额外射出雾化子弹。1 链技能 CD -1 回合，5 链潜行时气动 +25%，6 链解放暴击 +8% + 重击穿门 +50%。B-Tier 数值保守，满链 ×4。循环：放分身 → 虚实之门 → 雾化子弹输出 → 潜行 buff。
