# 菲比 移植计划

> 状态: ✅ 已实装
> 官方资料仅作参考,以本文件为最终设计依据
> 强度 Tier: **S**
> 移植级别: A → 工厂（重击型 + 双形态 · customLines + chainEffects）
> 定位: 衍射 · 音感仪 · 主C · **「赦罪/告解双形态 · 光噪效应」**

## 模拟器核心

菲比是双形态切换型主C。FORTE state toggle 管理形态切换——赦罪形态偏爆发（解放 +225%，重击星辉 +91%），告解形态偏持续输出（光噪效应加深，FFF 诅咒）。重击负责切换形态和输出。6 链镜之环召唤提供控制（停滞）+ 额外星辉。循环：赦罪输出 → 重击切换告解 → 光噪叠层 → 解放爆发。

## 模拟器保留

| 系统 | 模拟器抽象 |
|---|---|
| **衍射形态 toggle** | FORTE state max 1 → toggleForm，赦罪 ↔ 告解 |
| **赦罪形态** | 解放伤害 +225%（1 链），重击星辉 +91%（3 链） |
| **告解形态** | 光噪效应加深 +120%（2 链），FFF 诅咒 |
| **重击·星辉** | hasHeavy: true，告解形态下倍率提升 |
| **镜之环（6 链）** | 召唤时 atk +10% + 停滞效果 |
| **4 链全队** | 普攻命中降低目标衍射抗性 10% |
