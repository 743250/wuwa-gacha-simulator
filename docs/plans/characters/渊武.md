# 渊武 移植计划

> 状态: ✅ 已实装
> 官方技能、共鸣链名称与机制以 `docs/sources/characters/individual/渊武.json` 为准；源 JSON 暂无 Lv90 面板时使用 `template.js` 独立面板
> 强度 Tier: **C**
> 移植级别: A → 工厂（customLines + chainEffects · C-Tier 数值最小化）
> 定位: 导电 · 臂铠 · 辅助 · **「雷之楔 · 全队防御光环」**

## 模拟器核心

渊武是 4★ 防御型辅助（C-Tier）。FORTE gauge「雷势」max 100 → enhancedBurst ×1.5。技能放置「雷之楔」在场——范围内全队防御 +32%（6 链），雷之楔协同攻击（3 链 def × 20% 额外伤害）。4 链解放给全队护盾（def × 200%），5 链雷之楔在场时解放 +50%。C-Tier 满链 ×3.5。循环：放雷之楔 → 全队防御光环 → 解放护盾。


## 模拟器保留

| 官方系统名 | 模拟器抽象 |
|---|---|
| **雷煌拳** | 普攻按导电伤害结算 |

| **驱雷** | 延奏削韧/导电辅助方向合并进辅助定位 |

## 模拟器舍弃/合并

| 官方机制 | 处理 |
|---|---|
| 雷之楔范围、协同频率、目标追踪 | 合并为 3 回合场地与标准技能/防御增益 |
| 防御倍率造成额外伤害 | 合并为 `skillDmg` 与 `teamDef` 标准 effect |
| 抗打断、削韧细节 | 不单独模拟，保留防御辅助身份 |
