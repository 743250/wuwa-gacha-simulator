# 卡卡罗 移植计划

> 状态: ✅ 已实装
> 官方资料仅作参考,以本文件为最终设计依据
> 强度 Tier: **A**
> 移植级别: A → 工厂（重击型 + customLines + chainEffects）
> 定位: 导电 · 长刃 · 主C · **「杀意 · Deathblade 杀戮武装」**

## 模拟器核心

卡卡罗是窗口爆发型主C。普攻/技能积攒杀意（FORTE state max 100），解放直接拉满进入 Deathblade 杀戮武装（burstWindow ×1.5，3 回合），期间普攻/技能 +50% 伤害，3 链额外导电 +25%。6 链重击「死告」额外召唤 2 个猎杀影协同攻击（每人 atk × 100% 导电）。循环：叠杀意 → 解放进 Deathblade → 窗口内爆发 → 死告收尾。

## 模拟器保留

| 系统 | 模拟器抽象 |
|---|---|
| **杀意** | FORTE state max 100，普攻 +5 / 技能 +15 / 解放 +100 |
| **Deathblade** | burstWindow ×1.5（普攻/技能），3 回合 |
| **3 链铁腕外交** | Deathblade 期间导电 +25% |
| **6 链死告** | 重击额外召唤 2 猎杀影（atk × 100% × 2） |
| **1 链** | 技能命中额外回 10 能量 |
| **4 链** | 延奏全队导电 +20% |

## 模拟器舍弃/合并
