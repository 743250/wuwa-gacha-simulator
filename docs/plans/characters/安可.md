# 安可 移植计划

> 状态: ✅ 已实装
> 官方资料仅作参考,以本文件为最终设计依据
> 强度 Tier: **A**
> 移植级别: **S**（专属状态机 · 失序值 / 黑咩大暴走）
> 定位: 热熔 · 音感仪 · 主C · **「失序值 · 白咩-黑咩双形态」**

## 模拟器核心

安可是双形态爆发型主C。普攻/技能/变奏积攒失序值（0-100），满 100 时重击改为白咩·失控之炎（×3.5 共鸣解放伤害）。共鸣解放进入黑咩大暴走形态（4 回合），期间普攻/技能 ×1.5，重击 ×1.8，失序值积攒额外 +10，满 100 重击改为黑咩·暴走之炎（×4.5）。循环：积失序值 → 特殊重击 → 解放进黑咩 → 黑咩内爆发 → 退出。

## 模拟器保留

| 系统 | 模拟器抽象 |
|---|---|
| **失序值** | `unit.encoreDisorder` 0-100，普攻 +20 / 技能 +35 / 变奏 +30 / 重击 +20 |
| **黑咩形态** | `unit.encoreBlackTurns` 4 回合，普攻/技能 ×1.5，重击 ×1.8 |
| **白咩·失控之炎** | 失序满时重击（非黑咩）：×3.5 共鸣解放伤害 |
| **黑咩·暴走之炎** | 失序满时重击（黑咩内）：×4.5 共鸣解放伤害 |
| **1 链** | 普攻热熔伤害 +3% ×4 层 = +12% |
| **4 链** | 重击后全队热熔 +20% / 30s |
| **6 链** | 黑咩期间每次伤害叠迷失羔羊，每层 atk +5% ×5 层 |
