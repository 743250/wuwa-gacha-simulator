# 守岸人 移植计划

> 状态: ✅ 已实装
> 官方资料仅作参考,以本文件为最终设计依据
> 强度 Tier: **SS**
> 移植级别: **S**（专属状态机·星域）
> 定位: 衍射 · 音感仪 · 辅助 · **「星域」全队治疗 + 暴击暴伤增益**

## 模拟器核心

守岸人是鸣潮首位限定治疗辅助，核心为「星域」——共鸣解放展开星域（3 回合），每回合全队回血 + 暴击 +20% + 暴伤 +30%。1 链延长星域 +2 回合 + 切人不消失，2 链全队攻击 +40%，4 链技能治疗 +70%，6 链变奏伤害 +500%。循环：技能攒协奏 → 解放展开星域 → 全队持续受益。

## 模拟器保留

| 系统 | 模拟器抽象 |
|---|---|
| **星域** | `shorekeeperStarfield()` → 全队 buff：每回合回血 + crate +20% / cdmg +30% / 3 回合 |
| **协奏** | FORTE gauge「协奏」max 100 → shorekeeperField 标识 |
| **1 链不语者假想** | 星域延长 +2 回合，变奏后不结束星域 |
| **2 链夜幕的赠予** | 星域期间全队 atk +40% |
| **4 链万物寂静满溢** | 技能治疗加成 +70% |
| **6 链我所驶向的新世界** | 变奏伤害 +500%（×6） |

## 模拟器舍弃/合并

- 3 链「无限正将我等待」解放回协奏 20 → 简化到 `burstEnergyRefund`
- 5 链「来自缄默的回声」普攻牵引范围 → 回合制下合并为 `normalSplit` 2（多打 1 个敌人）
- 原版星域有多层 buff 叠层 → 合并为开星域时一次性上全 buff
