# 布兰特 移植计划

> 状态: ✅ 已实装
> 官方资料仅作参考,以本文件为最终设计依据
> 强度 Tier: **S**
> 移植级别: A → 工厂（customLines + chainEffects，不写专属 helper）
> 定位: 热熔 · 迅刀 · 辅助 · **「空中攻击 · 火焰归亡曲 · 全队治疗」**

## 模拟器核心

布兰特是带治疗/护盾的辅助型角色，核心为空中攻击派生 + 火焰归亡曲。变奏入场获得 3 层「乘风」（每层 +20% 伤害），叠满 FORTE「航路」后共鸣解放释放火焰归亡曲——AOE 伤害 + 全队治疗。4 链护盾量 +20% + 全队回血，6 链解放后再燃（火焰归亡曲 30% 额外热熔伤害）。循环：变奏入场叠乘风 → 叠航路 → 火焰归亡曲治疗 + 伤害 → 再燃。

## 模拟器保留

| 系统 | 模拟器抽象 |
|---|---|
| **乘风** | 变奏入场获 3 层（每层 +20% 伤害），叠满 +60% |
| **航路** | FORTE gauge max 100 → enhancedBurst ×1.6 |
| **火焰归亡曲** | 满航路解放：AOE + 全队 atk × 150% 治疗 |
| **2 链爆炸** | 延奏后下个角色技能触发爆炸 atk × 440% 热熔 |
| **4 链治疗** | 护盾量 +20% + 解放全队回血 |
| **6 链再燃** | 解放后额外 30% 火焰归亡曲热熔伤害 |

## 模拟器舍弃/合并

- 原版空中攻击多段连击 → 合并为乘风 buff 叠层
- 2 链延奏爆炸（atk × 440%）→ 合并到 burstDmg +42%
- 5 链普攻 +15% → 合并到 normalDmg chain effect
