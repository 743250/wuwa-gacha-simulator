# 珂莱塔 移植计划

> 状态: ✅ 已实装
> 官方资料仅作参考,以本文件为最终设计依据
> 强度 Tier: **S**
> 移植级别: A → 工厂（重击型 + customLines + chainEffects）
> 定位: 冷凝 · 佩枪 · 主C · **「晶体层数 → 致死以终 · 死兆爆发」**

## 模拟器核心

珂莱塔是叠层爆发型主C。共鸣技能积晶体层数（FORTE stacks max 5），满层强化共鸣技能「示我璀璨」（×2.0 冷凝爆发）。对目标附加解离/变彩状态提升暴击率。重击「末路见行」（5 链 +47%）和共鸣解放二段「死兆」（6 链 +186.6%）为核心爆发点。循环：叠晶体 → 示我璀璨 → 末路见行 → 致死以终 → 死兆。

## 模拟器保留

| 系统 | 模拟器抽象 |
|---|---|
| **晶体层数** | FORTE stacks max 5 → enhancedSkill ×2.0 |
| **解离/变彩** | 技能命中附加 debuff（1 链暴击 +12.5% 对解离目标） |
| **末路见行（重击）** | hasHeavy: true，5 链 +47% |
| **致死以终 → 死兆** | 共鸣解放 AOE（6 链死兆 +186.6%） |
| **4 链全队技能** | 重击后全队共鸣技能 +25% |

## 模拟器舍弃/合并

- 原版灵萃/晶体/变彩三状态 → 合并为 FORTE stacks + 解离 debuff
- 原版死兆射击晶体翻倍 → 简化为 6 链 burstDmg +186.6%
- 3 链延奏碎璃镜花 → 合并到 skillDmg 中
