## 模拟器核心

维里奈是常驻 5★ 治疗辅助。核心为「光合标记」——共鸣技能给敌人挂标记（FORTE gauge），队友攻击标记目标时触发全队治疗（atk × 30%）+ 衍射伤害 +10%。共鸣解放「草木生长」全队大治疗 + 光合标记持续效果。1 链延奏后登场角色持续回血，2 链技能额外回光合能量 + 协奏，6 链重击星星花绽放协同攻击。循环：挂光合标记 → 队友攻击触发治疗 → 解放全队大治疗。

## 模拟器保留

| 系统 | 模拟器抽象 |
|---|---|
| **光合标记** | FORTE gauge「光合」max 100 → healField ×0.8（满时展开 3 回合持续治疗） |
| **星星花绽放** | 重击/空中攻击触发协同攻击（6 链 ×20% 伤害 + 回血） |
| **1 链萌芽** | 延奏后登场角色持续回血（atk × 20%/5s） |
| **4 链盛放** | 重击/解放/延奏后全队衍射 +15% |
| **5 链结果** | 治疗低 HP 角色时治疗 +20% |

## 模拟器舍弃/合并

- 原版「光合能量」「念意」多资源 → 合并为 FORTE gauge
- 原版草木生长 + 频隙回响双治疗 → 合并为解放全队大治疗
- 3 链「生长的选择」光合标记治疗 +12% → chainEffects `heal` 0.12
