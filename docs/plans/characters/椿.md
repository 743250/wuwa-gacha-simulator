# 椿 移植计划

> 状态: ✅ 已实装
> 官方资料仅作参考,以本文件为最终设计依据
> 强度 Tier: **S**
> 移植级别: A → 工厂（customLines + chainEffects，不写专属 helper）
> 定位: 湮灭 · 迅刀 · 主C · **「红椿蕊 → 含苞强化形态 · 一日花爆发」**

## 模拟器核心

椿是双资源管理型主C。普攻/技能积攒红椿（gauge 0-100），满红椿时共鸣技能替换为一日花——消耗全部红椿进入含苞状态。含苞期间攻击 +58%（3 链），酣梦倍率提升。6 链永生花含苞期间协奏满时可再释放一日花（100% 伤害）。循环：叠红椿 → 一日花进含苞 → 含苞内爆发 → 退出。

## 模拟器保留

| 系统 | 模拟器抽象 |
|---|---|
| **红椿** | FORTE gauge「红椿」max 100 → enhancedNormal ×1.7 |
| **含苞状态** | 满红椿释放一日花后进入，atk +58%（3 链），酣梦强化 |
| **一日花** | 满红椿时技能强化（2 链 +120% 倍率） |
| **3 链含苞攻击** | 含苞期间 atk +58%（`atk` value 0.58） |
| **4 链全队普攻** | 变奏后全队普攻 +25%（`teamNormalDmg` value 0.25） |
| **6 链永生花** | 酣梦倍率额外 +150%，含苞 ×1.5 → ×2.5 |
