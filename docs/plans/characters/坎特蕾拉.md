# 坎特蕾拉 移植计划

> 状态: ✅ 已实装
> 官方资料仅作参考,以本文件为最终设计依据
> 强度 Tier: **S**
> 移植级别: A → 工厂（customLines + chainEffects，不写专属 helper）
> 定位: 湮灭 · 音感仪 · 副C · **「迷离 → 蜃境 → 惊醒」三层递进机制**

## 模拟器核心

坎特蕾拉是三层递进机制型副C。迷离（FORTE gauge 0-100）：普攻/技能积攒，满时进入蜃境——解放伤害 ×1.8，治疗加成 +25%（4 链）。2 链解放给主目标挂「迷梦」，目标受到非迷梦伤害时触发「惊醒」额外湮灭伤害。1 链技能伤害 +50% 免打断，3 链解放后直接进入蜃境。循环：叠迷离 → 进蜃境 → 解放迷梦标记 → 全队攻击目标惊醒爆发。

## 模拟器保留

| 系统 | 模拟器抽象 |
|---|---|
| **迷离** | FORTE gauge max 100 → enhancedBurst ×1.8 |
| **蜃境** | 迷离满时进入，解放伤害提升，4 链治疗 +25% |
| **迷梦** | 解放给主目标挂 debuff（2 链惊醒倍率 +245%） |
| **惊醒** | 迷梦目标受非迷梦伤害触发额外湮灭伤害 |
| **1 链感知汲取** | 技能伤害 +50% 免打断 |
| **6 链蛰幻** | 普攻 +80% + 解放无视 30% 防御 |

## 模拟器舍弃/合并

- 原版织梦水母（召唤物 5 只）→ 合并到蜃境 buff
- 原版迷梦前 1.2s 不触发惊醒 → 回合制下简化为受非迷梦伤害即触发
- 5 链弥漫水母 +5 → 合并为 burstDmg +50%
