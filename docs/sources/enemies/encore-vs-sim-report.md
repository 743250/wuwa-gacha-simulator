## 8. 推荐改造方案

### 8.1 现状
- 模拟器 enemies.js 共 43 个敌人：17 世界 BOSS + 6 剧情/周本 BOSS + 7 Common + 7 Elite + 3 小怪 + 3 召唤物
- API 有 269 个敌人，远超模拟器规模
- 已匹配 23 个，可直接利用 API 校准数值

### 8.2 优先级
1. **立即**：对已匹配的 23 个敌人，用 API 的 Lv90 数值替换 enemies.js 中的手动估算值
2. **高优先级**：将 Common（轻波级）和 Elite（巨浪级）敌人按需扩充到副本池（200 个可选）
3. **中优先级**：补全缺失的 24 个 Overlord 级 BOSS（芬莱克、共鸣回响·芬莱克、共鸣回响·梦魇亚当·重锤、海维夏、海维夏…）
4. **低优先级**：15 个 Calamity 级 BOSS（阿列夫一造物、达妮娅、芙露德莉斯…），周本/剧情为主，暂不急

### 8.3 数据迁移策略
1. 在 enemies.js 中每个敌人增加 `encoreId` 字段指向 API 敌人 ID
2. 用 `spawnEnemy()` 从紧凑 JSON 读 Lv90 数值
3. 抗性字段：API 用千分比（1000=10%），需除以 10000 转为小数
4. 保留模拟器自定义的 mechanic/description 字段
