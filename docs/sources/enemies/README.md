## 已知差距

1. ⚠️ **HP 值为深塔缩放后数据**，非世界 BOSS 基础 HP。官方世界 BOSS 会根据玩家世界等级（SOL3）动态缩放
2. ⚠️ **推荐等级**（minLevel）无官方数据——官方不使用固定推荐等级，BOSS 随世界等级缩放
3. ⚠️ 部分 v3.x 新 BOSS（如 Lampylumen Myriad）未收录，待后续版本补充
4. ⚠️ 深塔敌人配置每 28 天轮换，上述为多个周期的聚合数据

## 模拟器折算说明

- 模拟器不实现 SOL3 世界等级系统
- BOSS 基础 HP 以深塔血量反推的相对值为准，再除以经验缩放系数
- 抗性模型已对齐官方：对自身元素 40% 抗性、其他元素 10% 抗性
- 深塔水温曲线（balance.js ABYSS_TEMPERATURE_TABLE）已按官方中一膨胀数据校准
