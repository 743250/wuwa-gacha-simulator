## 技能组

### 重器藏锋

```
普攻
进行最多4段的连续攻击，造成湮灭伤害。

重击
消耗耐力，造成湮灭伤害。
若长按普攻，则进入磐岩守势状态。

磐岩守势
桃祈受到的伤害降低35%；
若磐岩守势期间桃祈受到攻击，将会自动施放后发制人；
磐岩守势持续3秒后，将会自动施放后发制人；
在施放共鸣技能磐岩护壁时若受到攻击，将会自动施放后发制人。

后发制人
攻击目标，造成湮灭伤害。

空中攻击
消耗耐力进行空中下落攻击，造成湮灭伤害。

闪避反击
成功闪避后一定时间内短按普攻，将会攻击目标，造成湮灭伤害。
```
