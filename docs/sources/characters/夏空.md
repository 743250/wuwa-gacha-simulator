## 技能组

### 

```
四拍的舞曲

普攻
进行最多4段的连续攻击，造成气动伤害。普攻第4段可为命中的目标添加一层【风蚀效应】。
·使用闪避打断普攻前三段时，一定时间内短按普攻，将会施放对应段数的普攻。
·普攻第4段结束后，夏空会进入音律独奏状态。夏空在主动或被动打断普攻第4段或音律独奏状态时，会生成一个合奏音影。

音律独奏
夏空或合奏音影在进入音律独奏状态时能够以自身为中心，使附近队伍中所有角色的气动伤害加成提升24%，该效果不可叠加。

合奏音影
场上最多存在2个合奏音影。
若夏空的普攻第4段被打断，合奏音影将代替夏空完成这次攻击，并进入音律独奏状态。
若夏空的音律独奏状态被打断，合奏音影将代替夏空继续音律独奏状态。
通过共鸣技能生成合奏音影时，除上述两种情况外，合奏音影将代替夏空完成这次攻击，但不进入音律独奏状态。

重击
消耗耐力并跃向空中，攻击目标，造成气动伤害。

瞄准攻击
短按瞄准键，进入瞄准状态，可以进行射击。
蓄力完成后攻击目标，造成气动伤害，此次伤害为重击伤害。

空中攻击
消耗耐力进行最多2段的攻击，造成气动伤害。
施放第2段空中攻击后一定时间内短按普攻，将会施放第4段普攻。

闪避反击
成功闪避时，一定时间内短按普攻，将会攻击目标，造成气动伤害。
```
