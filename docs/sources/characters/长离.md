## 技能组

### 衔火洞明

```
普攻
进行最多4段的连续攻击，造成热熔伤害。
施放地面普攻第4段后进入心眼状态，持续12秒。

空中攻击
消耗耐力，在空中进行最多4段的连续攻击，造成热熔伤害。
施放空中攻击第4段后进入心眼状态，持续12秒。

重击
在地面长按普攻，消耗耐力上挑攻击目标，造成热熔伤害。一定时间内短按普攻将会施放空中攻击第3段。

空中重击
在空中长按普攻或施放普攻心眼·冲后一定时间内短按普攻，消耗耐力进行空中下落攻击，造成热熔伤害。一定时间内短按普攻将会施放普攻第3段。

闪避反击
成功闪避后一定时间内短按普攻，将会攻击目标，造成热熔伤害。
```
