## 技能组

### 真源构演

```
普攻

进行最多4段的连续攻击，造成衍射伤害。攻击命中目标时生成坍缩核。

重击

长按普攻持续消耗耐力，进入解限状态，在该状态下：
每1秒积蓄1段演算数据。
可以自动采集一定范围内的植物类采集物。
耐力不足或通过点按普攻等方式结束解限状态时，对目标造成衍射伤害，同时已积累的每1段演算数据转化为1段【实论数据】并生成1枚坍缩核。

空中攻击
消耗耐力进行空中下落攻击。攻击命中目标时生成1枚坍缩核。一定时间内短按普攻将会施放普攻第2段。

闪避反击
成功闪避后一定时间内短按普攻，将会攻击目标，造成衍射伤害。
```
