## 技能组

### 常态攻击: 船长的狂想演出

```
普攻
进行最多4段的连续攻击，造成热熔伤害。

重击
消耗耐力攻击目标，造成热熔伤害。

重击·狂想即兴
消耗耐力攻击目标，造成热熔伤害。
·施放普攻第2段或第4段后，长按普攻，可以释放重击·狂想即兴。
·施放空中攻击第4段后，短按普攻，可以施放重击·狂想即兴。

空中攻击
进行最多4段连续攻击，造成热熔伤害。空中攻击第1段或第2段按下普攻时，布兰特向目标发起钩索冲荡，触及目标后会向目标发起攻击，若在动作持续期间松开普攻，则会在攻击命中目标后向后空翻；若持续按住，则会持续攻击目标，持续时间结束后，向后空翻。空中攻击第3段攻击后会向后空翻。空中攻击每次空翻后，按下普攻可施放下一段空中攻击，空翻可重置空中闪避次数。
·若空中攻击第1段钩索冲荡未触及目标，则向前进行一次斩击，造成热熔伤害。
·若空中攻击第1段或第2段钩索冲荡触及目标，但是发起攻击未命中目标，则会短暂悬停后下落。

闪避反击
成功闪避后，一定时间内短按普攻，将会攻击目标，造成热熔伤害。
```
