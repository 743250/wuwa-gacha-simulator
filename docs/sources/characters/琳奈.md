```
泛彩流光

普攻
短按或长按施放普攻，进行最多3段的连续攻击，造成衍射伤害。

空中攻击
消耗耐力进行空中下落攻击，造成衍射伤害。

闪避反击
成功闪避后一定时间内短按普攻，将会攻击目标，造成衍射伤害。

普攻·灵感碰撞
处于「光学取样」阶段时，如果【溢彩】已满，按住普攻进入蓄力状态。
蓄力期间，每0.2秒将15点【溢彩】转换为12.5%的【流光】，蓄力期间免疫打断且受到伤害降低50%。
松开普攻或【流光】满时，琳奈将根据当前转换的【流光】百分比造成衍射伤害：
普攻·灵感碰撞·1级：当前【流光】<50%
普攻·灵感碰撞·2级：50% &le;当前【流光】<100%
普攻·灵感碰撞·3级：当前【流光】=100%

绮彩巡游·普攻
绮彩巡游状态期间，普攻替换为绮彩巡游·普攻。
进行最多5段的连续攻击，造成衍射伤害。
【流光】达到120点时，绮彩巡游·普攻第1段和绮彩巡游·普攻第4段的追击能力提升。

绮彩巡游·地面重击
绮彩巡游状态期间，且处于地面时，按住普攻持续施放。琳奈持续消耗耐力攻击目标，造成衍射伤害，此次伤害为普攻伤害；松开普攻时，琳奈施放绮彩巡游·跃动集束，造成衍射伤害，此次伤害为普攻伤害。

绮彩巡游·空中重击
绮彩巡游状态期间，且处于空中时，长按普攻施放。消耗耐力攻击目标，造成衍射伤害，此次伤害为普攻伤害。
处于空中时无法刷新该技能释放次数，琳奈为登场角色且处于地面时刷新可释放次数。

绮彩巡游·空中攻击
绮彩巡游状态期间，空中攻击替换为绮彩巡游·空中攻击。
消耗耐力进行空中下落攻击，造成衍射伤害。

绮彩巡游·闪避反击
绮彩巡游状态期间，闪避反击替换为绮彩巡游·闪避反击。
成功闪避后一定时间内短按普攻，将会攻击目标，造成衍射伤害。
施放该技能后，若琳奈处于绮彩巡游，一定时间内短按普攻将会施放绮彩巡游·普攻第2段。
```
