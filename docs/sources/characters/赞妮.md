## 技能组

### 黯夜将明

```
烈阳余烬
赞妮处于编队时，附近队伍中的角色对目标附加【光噪效应】时，将立刻结算全部层数伤害并转化为相同层数的【烈阳余烬】。每次发生转化，赞妮根据转化层数汲取【焰光】。【烈阳余烬】层数上限为60层，每层持续6秒。该层数可触发「此间永驻之光」 合鸣效果。

引燃明灯
赞妮处于灼焰形态时，解锁重斩·破晓、重斩·将明、重斩·终夜、重斩·闪裂，造成衍射伤害，此次伤害为重击伤害，同时视为【光噪效应】伤害。
【焰光】不少于30点时，共鸣技能标准防卫预案替换为共鸣技能引燃明灯。
按住共鸣技能可进入准备架势，准备架势期间免疫打断，若切换至其他角色则提前退出。准备架势期间松开共鸣技能，可施放重斩·破晓并消耗【焰光】。
若进入准备架势一定时间内受到敌方目标攻击，此次伤害降低100%同时停滞周围的目标，可施放重斩·闪裂并消耗【焰光】，额外削减其10%共振度。赞妮在2秒内受到伤害降低30%。施放重斩·闪裂后，普攻替换为重斩·终夜。
重斩·终夜命中时可以消耗赞妮至多40点【焰光】，每消耗1点【焰光】，使本次重斩·终夜的伤害倍率额外增加。施放第1段重斩·终夜期间若被打断，可继续短按普攻施放第2段重斩·终夜。
准备架势持续时间结束后，直接施放重斩·终夜。
【焰光】不少于30点时，普攻替换为重斩·破晓。进入灼焰形态后，若立即施放普攻，此次普攻替换为共鸣技能引燃明灯，若按住普攻可进入准备架势。
施放重斩·破晓后若立即施放普攻，可施放普攻重斩·将明并消耗【焰光】。
施放重斩·将明后，普攻替换为重斩·终夜。
成功闪避后，若此时普攻没有替换为重斩·终夜且【焰光】不少于30点，则一定时间内短按普攻，可施放重斩·闪裂并消耗【焰光】。

冗余动能获取规则
【冗余动能】上限100点。
常态攻击命中目标时，获得【冗余动能】。
施放变奏技能即刻执行时，获得【冗余动能】。
施放共鸣技能标准防卫预案时，获得【冗余动能】。
精准反击命中目标时，获得【冗余动能】。
处于灼焰形态时，无法获得【冗余动能】。

焰光获取规则
赞妮未处于灼焰形态时，【焰光】上限为100点。
赞妮处于灼焰形态时，【焰光】上限为150点。
施放集中压制或破袭反击后，获得10点【焰光】。
每1层【光噪效应】转化为【烈阳余烬】时，获得5点【焰光】。
施放共鸣解放重燃时，获得50点【焰光】。
```
