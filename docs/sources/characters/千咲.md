## 技能组

### 俱寂

```
普攻
进行最多2段的连续攻击，造成湮灭伤害。
施放普攻第2段后在一定时间内短按普攻，将会施放纷裂贯穿。

纷裂贯穿
造成湮灭伤害。
在地面施放该技能后一定时间内短按普攻，将会施放断命之铗。
在空中施放该技能后一定时间内按下普攻，将会施放垂落终焉。

断命之铗
张开剪刀，造成湮灭伤害，一段时间后剪出，造成湮灭伤害，回复附近队伍中所有角色的生命值。
施放该技能期间按下普攻，可额外造成湮灭伤害，再次按下普攻，将直接剪出。
该技能伤害为共鸣解放伤害。
断命之铗剪出后一定时间内短按普攻，将会施放撤弦。

撤弦
造成湮灭伤害。
施放该技能一定时间内，可以触发成功闪避效果。

重击
跃至空中，消耗耐力攻击目标，造成湮灭伤害。
处于电锯模式时无法施放该技能。
施放该技能后进行下述操作，将会施放垂落终焉：
·一定时间内短按普攻。
·一定时间后施放命弦·掠行，在落地前下一次短按普攻。
施放重击至落地期间，下一次长按普攻，将会施放重击·碎响切面。

空中攻击
消耗耐力进行空中下落攻击，造成湮灭伤害。
不处于电锯模式时，施放该技能后一定时间内短按普攻，将会施放普攻第2段。

重击·碎响切面
造成湮灭伤害。
施放重击·碎响切面后一定时间内按下普攻，将会施放垂落终焉。
施放重击·碎响切面期间，若被打断，则在落地前，下一次长按普攻，可再次施放重击·碎响切面，落地前无法重复触发该效果。

垂落终焉
消耗耐力进行空中下落攻击，造成湮灭伤害。
在地面施放该技能后一定时间内短按普攻，将会施放断命之铗。

闪避反击
成功闪避后一定时间内短按普攻，将会攻击目标，造成湮灭伤害。
施放该技能后一定时间内短按普攻，将会施放纷裂贯穿。

闪避反击·解弦之眼·收弦
在地面时，成功闪避后保持按住闪避，将会施放闪避反击·解弦之眼·收弦攻击目标，停滞目标，造成湮灭伤害。
·不处于电锯模式时，施放该技能后，下次在地面短按普攻或在空中按下普攻，将会施放纷裂贯穿，进入电锯模式时，移除该效果。
·处于电锯模式时，施放该技能后一定时间内短按普攻，将会施放锯环·疾攻第3段。
```
