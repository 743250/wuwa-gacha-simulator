```
普攻

消耗耐力攻击目标，造成导电伤害。施放后短时间内短按普攻，可以施放普攻第2段。

空中攻击
消耗耐力进行空中下落攻击，造成导电伤害。

闪避反击
成功闪避后一定时间内短按普攻，将会攻击目标，造成导电伤害。

空中闪避反击
空中成功闪避后一定时间内短按普攻，消耗耐力进行空中下落攻击，造成导电伤害。

重击·烁雷·后撤
【战势】满时，重击·鸣铁替换为重击·烁雷·后撤，造成导电伤害，施放时消耗全部的【战势】。
·技能施放时，一定时间后松开普攻，或者再次短按普攻，可以施放重击·烁雷·旋切。
·技能被打断后一定时间内若没有离开地面，亦可施放重击·烁雷·旋切。

重击·烁雷·旋切
奥古斯塔舞剑旋转，造成导电伤害。

重击·烁雷·升拳
当施放共鸣技能·斩锋时，若【战势】满，短时间内短按共鸣技能可以施放重击·烁雷·升拳，造成导电伤害，施放时消耗全部的【战势】。
当施放重击·鸣铁时，若【战势】满，短时间内短按普攻可以施放重击·烁雷·升拳，造成导电伤害，施放时消耗全部的【战势】。

闪避反击·重击·鸣铁

·技能施放时，一定时间后松开普攻，或者再次短按普攻，可以施放重击·烁雷·旋切。
·技能被打断后一定时间内若没有离开地面，亦可施放重击·烁雷·旋切。

【战势】获取规则
【战势】上限为100点。
·普攻、重击·鸣铁、空中攻击、共鸣技能·斩锋，闪避反击，闪避反击·重击·鸣铁造成伤害时可以获得【战势】。
```
