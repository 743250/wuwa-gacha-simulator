## 技能组

### 常态攻击: 生与死的乐章

```
普攻
进行最多3段的连续攻击，造成湮灭伤害。
施放普攻第3段时进入重世状态。

重击
消耗耐力攻击目标，造成湮灭伤害。
施放该技能后在一定时间内短按普攻，将会施放普攻第2段。

谱曲终末
弗洛洛每25秒可激活谱曲。
同时满足下述3种条件时，重击替换为谱曲终末。
·拥有6枚【乐声】。
·谱曲处于激活状态。
·不处于【定音】状态。

施放谱曲终末，消耗耐力，停滞并牵引附近目标，造成湮灭伤害，本次伤害为共鸣技能伤害。
每层余响可使本次伤害倍率额外增加。
施放此技能时，同时视为施放声骸技能。
施放此技能时，谱曲进入冷却状态，随后获得【定音】状态。

空中攻击
消耗耐力进行空中下落攻击，造成湮灭伤害。

闪避反击
成功闪避后一定时间内短按普攻，将会攻击目标，造成湮灭伤害。
施放该技能后在一定时间内短按普攻，将会施放普攻第3段。
```
