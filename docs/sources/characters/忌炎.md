## 技能组

### 孤枪止戈

```
普攻

进行至多5段的连续攻击，造成气动伤害。

重击

消耗耐力进行突刺攻击，造成气动伤害。

重击·凌空击

若在重击期间按住普攻，会在重击结束后施放凌空击，造成气动伤害。

重击·潜渊斩

若在重击期间松开普攻，会在重击结束后施放潜渊斩，造成气动伤害。

空中攻击

消耗耐力进行空中下落攻击，造成气动伤害。

在下落攻击后一定时间内短按普攻，将会追击目标，造成气动伤害。

空中攻击·长空不坠

施放重击凌空击或在空中施放共鸣技能枪扫风定后，可以进行浮空攻击，造成气动伤害。

闪避反击

成功闪避后一定时间内短按普攻，将会攻击目标，造成气动伤害。
```
