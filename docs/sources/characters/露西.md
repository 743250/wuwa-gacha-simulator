## 技能组

### 靶定线程

```
普攻
进行最多4段的连续攻击，造成衍射伤害。
施放普攻第1段后，长按普攻，将会持续攻击目标，造成衍射伤害。持续长按一定时间后将会施放重击第1段。
普攻第3段命中目标且在施放后一定时间内长按普攻将会施放重击第2段。

重击
消耗耐力，甩出分子线造成衍射伤害。
重击第1段命中目标且持续长按普攻，将会施放重击第2段，造成衍射伤害。
重击第1段施放后一定时间内短按普攻将会施放普攻第3段。

空中攻击
消耗耐力进行空中下落攻击，造成衍射伤害。
施放该技能后一定时间内短按普攻将会施放普攻第2段。

闪避反击
成功闪避后一定时间内短按普攻，将会攻击目标，造成衍射伤害。
施放该技能后一定时间内短按普攻将会施放普攻第4段。
该技能命中目标后一定时间内长按普攻将会施放重击第2段。

普攻·线程撕裂
处于算法压缩状态时，普攻替换为普攻·线程撕裂。
进行最多4段的连续攻击，造成衍射伤害，本次伤害为重击伤害。

重击·单线程
处于算法压缩状态时，重击替换为重击·单线程。
消耗耐力攻击目标，造成衍射伤害。
施放该技能后一定时间内短按普攻且自身处于空中时，将会施放空中攻击·算法压缩。
施放该技能后一定时间内短按普攻且自身处于地面时，将会施放普攻·线程撕裂第2段。

重击·双线程
处于算法压缩状态时且【根权限】达到100点时重击·单线程替换为重击·双线程。
消耗耐力和所有【根权限】攻击目标，造成衍射伤害。
施放该技能后一定时间内长按或短按普攻且自身处于空中时，将会施放重击·多线程。
施放该技能后一定时间内短按普攻且自身处于地面时，将会施放普攻·线程撕裂第2段。

重击·多线程
消耗耐力攻击目标，造成衍射伤害并附加【骇破·偏移】。施放时若自身持有SQL，则使本次伤害倍率提升270%并移除SQL。
施放该技能后一定时间内短按普攻将会施放普攻·线程撕裂第2段。

空中攻击·算法压缩
处于算法压缩状态时，空中攻击替换为空中攻击·算法压缩。
消耗耐力进行空中下落攻击，造成衍射伤害。
施放该技能后一定时间内短按普攻将会施放普攻·线程撕裂第2段。

闪避反击·算法压缩
处于算法压缩状态时，闪避反击替换为闪避反击·算法压缩。
成功闪避后一定时间内短按普攻，将会攻击目标，造成衍射伤害。
施放该技能后一定时间内短按普攻将会施放普攻·线程撕裂第3段。
```
