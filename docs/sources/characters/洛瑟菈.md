## 技能组

### 

```
抓拍

普攻
进行最多3段的连续攻击，造成冷凝伤害。
施放普攻第3段时，长按普攻，展开聚焦环，指针将在聚焦环上移动。
·若指针不在完美焦距时抬起普攻，洛瑟菈将施放普攻第3段·乏善可陈，造成冷凝伤害并回复【印象】。
·若指针在完美焦距时抬起普攻，洛瑟菈则将施放普攻第3段·可圈可点，造成更高的冷凝伤害并回复更多的【印象】。
若施放普攻第2段后一定时间内短按普攻，则不会展开聚焦环，并直接施放普攻第3段·乏善可陈。
展开聚焦环期间，完美焦距区域将逐渐扩大。完美焦距区域填满聚焦环后则自动收起，洛瑟菈将直接施放普攻第3段·可圈可点。
长按普攻时，可依次施放普攻第1、2、3段，且施放普攻第3段时将展开聚焦环。

空中攻击
消耗耐力进行空中下落攻击，造成冷凝伤害。
施放该技能后在一定时间内短按普攻将会施放普攻第2段。

闪避反击
成功闪避后一定时间内短按普攻，将会攻击目标，造成冷凝伤害。
施放该技能后在一定时间内短按普攻将会施放普攻第3段。
```
