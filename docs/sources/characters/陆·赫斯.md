## 技能组

### 光犹如此

```
普攻
进行最多4段的连续攻击，造成衍射伤害。
第3段攻击会发出飞刃，施放其他可造成伤害的技能或陆·赫斯处于后台时，飞刃会消失。

重击
消耗耐力跳至空中，造成衍射伤害。

空中攻击
在空中进行最多4段的连续攻击，造成衍射伤害，第4段为消耗耐力进行空中下落攻击。
空中的第2、3段普攻，可以通过短按普攻或短按跳跃施放。
通过短按普攻施放时，使用镰刀·剖解进行攻击。
通过短按跳跃施放时，使用镰刀·裁断进行攻击，对命中目标附加【集谐·偏移】，持续25秒，并使自己在1秒内受到的伤害降低15%。
作为登场角色在空中时，该技能连击段数不会被重置。
在空中时，长按普攻，消耗耐力进行空中下落攻击。

闪避反击
在地面成功闪避后一定时间内短按普攻，将造成衍射伤害。施放该技能后一定时间内短按普攻，将施放普攻第3段。
在空中成功闪避后一定时间内短按普攻，将会高速位移至目标位置，造成衍射伤害。
```
