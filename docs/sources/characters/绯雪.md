## 技能组

### 灼樱刀术

```
普攻·常世身
连续短按或按住普攻，进行最多3段的连续攻击，造成冷凝伤害。
普攻·常世身第3段会为命中的目标附加1次【霜渐效应】。

重击·寒簇·常世身
【心念】满300点时，解锁重击·寒簇·常世身。
长按普攻，消耗耐力施放，期间免疫伤害和受击。绯雪依次射出三支箭矢，造成冷凝伤害，此次伤害为共鸣解放伤害。
射出最后一支箭矢时，消耗300点【心念】，解锁预求我身·见心，该箭矢命中目标时，为命中的目标附加1次【霜渐效应】。

空中攻击·常世身
消耗耐力进行空中下落攻击，造成冷凝伤害。

闪避反击·常世身
成功闪避后一定时间内短按普攻，将会攻击目标，造成冷凝伤害。
施放该技能后在一定时间内短按普攻，将会施放普攻·常世身第3段。

普攻·预求身
预求身期间，普攻·常世身替换为普攻·预求身。
进行最多5段的连续攻击，造成冷凝伤害，此次伤害为共鸣解放伤害。
普攻·预求身第3、4、5段会为命中的目标附加1次【霜渐效应】。

重击·预求身
预求身期间，按住普攻，将消耗耐力，进入【沉息】，【沉息】期间持续消耗耐力。松开普攻或耐力耗尽时，绯雪退出【沉息】，挥刃向前突进，造成冷凝伤害，此次伤害为共鸣解放伤害。施放该技能后在一定时间内短按普攻，将会施放普攻·预求身第2段。
若挥刃向前突进的一定时间内受到敌方目标攻击，免疫此次伤害并停滞周围敌人，随后一定时间内免疫打断且受到的伤害降低100%，一定时间内短按普攻改为衔接普攻·预求身第4段。

重击·枯霜·预求身
【淬寒·枯霜】满时，重击·预求身替换为重击·枯霜·预求身。
长按普攻，消耗耐力，施放期间消耗3层【淬寒·枯霜】，获得1点【锻雪·归刃】，造成冷凝伤害，此次伤害为共鸣解放伤害。
重击·枯霜·预求身会为命中的目标附加1次【霜渐效应】。
绯雪处于空中时，若施放该技能，绯雪会先回到地面。

空中攻击·预求身
预求身期间，空中攻击·常世身替换为空中攻击·预求身。
消耗耐力，在空中进行最多3段的连续攻击，造成冷凝伤害，此次伤害为共鸣解放伤害，第3段为空中下落攻击。
空中攻击·预求身第2、3段会为命中的目标附加1次【霜渐效应】。
施放空中攻击·预求身一定时间后，在空中施放除普攻·居合以外的技能时，空中攻击·预求身的连击段数不会被重置。
在空中时，长按普攻，消耗耐力施放空中下落攻击。

闪避反击·预求身
预求身期间，闪避反击·常世身替换为闪避反击·预求身。
成功闪避后一定时间内短按普攻，将会攻击目标，造成冷凝伤害，此次伤害为共鸣解放伤害。
施放该技能后在一定时间内短按普攻，将会施放普攻·预求身第3段。
```
