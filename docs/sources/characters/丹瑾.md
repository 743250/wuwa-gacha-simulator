### 执刃

```
普攻

进行最多3段的连续攻击，造成湮灭伤害。

重击

消耗耐力攻击目标，发动连续攻击，造成湮灭伤害。

空中攻击

消耗耐力进行空中下落攻击，造成湮灭伤害。

闪避反击

成功闪避后一定时间内短按普攻，将会攻击目标，造成湮灭伤害。

闪避反击·逐影

闪避反击成功后，丹瑾可使用共鸣技能朱华残章，施放共鸣技能朱蚀。
```
