### 常态攻击: 虚像陷落

```
普攻
进行最多3段的连续攻击，造成湮灭伤害。

重击
消耗耐力攻击目标，造成湮灭伤害。

重击·浮潜幻海
拥有【迷离】时，重击替换为浮潜幻海，造成湮灭伤害，并进入蜃境状态。若当前已处于蜃境状态，施放浮潜幻海不会重新进入蜃境状态。
可在水中施放。

空中攻击
消耗耐力进行空中下落攻击，造成湮灭伤害。

闪避反击
成功闪避后一定时间内短按普攻，将会攻击目标，造成湮灭伤害。
```
