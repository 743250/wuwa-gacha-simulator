## 技能组

### 佩洛，悠着点

```
普攻
进行最多4段的连续攻击，造成湮灭伤害。

重击
消耗耐力，攻击目标，造成湮灭伤害，命中目标时若【想象力】至少有100点，则将洛可可送上空中，进入飞跃幻想状态。可通过持续按住普攻进行蓄力，蓄力越久能获得越多的【想象力】，蓄力期间若耐力耗尽亦可继续蓄力。蓄力中途松开普攻，或者达到蓄力的最大时间时，会自动施放重击。

空中攻击
消耗耐力进行空中下落攻击，造成湮灭伤害。

闪避反击
成功闪避后一定时间内短按普攻，将会攻击目标，造成湮灭伤害。
```
