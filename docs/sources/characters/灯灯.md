## 技能组

### 常态攻击: 前路领航

```
黄灯·普攻
召唤啾啾进行三连射击，造成导电伤害。

黄灯·加速
短按闪避，可施放加速，并进入疾奔状态，加速无法成功闪避敌人的攻击。

黄灯·疾奔
该状态下，自动释放飞星攻击被锁定的敌人，造成导电伤害，此次伤害为普攻伤害。

黄灯·下落攻击
消耗耐力进行空中下落攻击，造成导电伤害。

红灯·普攻
进行最多3段的连续攻击，造成导电伤害。

红灯·重击
消耗耐力，啾啾和灯灯一同下砸，攻击目标，造成导电伤害，此次伤害为普攻伤害。

红灯·下落攻击
消耗耐力进行空中下落攻击，造成导电伤害。

红灯·闪避反击
成功闪避后一定时间内短按普攻，攻击目标，造成导电伤害。
```
