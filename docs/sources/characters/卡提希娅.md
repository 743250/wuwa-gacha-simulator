### 常态攻击: 以剑奉读此身

```
普攻·卡提希娅
进行最多4段的连续攻击，造成气动伤害。
施放第4段普攻·卡提希娅后，会为命中的目标附加1层【风蚀效应】，同时会召唤【神权剑之影】。
【神权剑之影】持续20秒，至多仅存在1把。

重击·卡提希娅
消耗耐力攻击目标，造成气动伤害，同时会召唤【异权剑之影】，可在空中施放。
【异权剑之影】持续20秒，至多仅存在1把。
此次伤害视为普攻伤害。

空中攻击·卡提希娅
空中松开普攻时，消耗耐力进行空中下落攻击，造成气动伤害，同时视为【风蚀效应】伤害。
施放下落攻击后一定时间短按普攻将会施放普攻·卡提希娅第2段。
施放空中攻击·卡提希娅后，卡提希娅回收附近场上所有【剑之影】，根据【剑之影】的种类之和，改变此次下落攻击的形态，并分别对应获得【人权之心】、【神权之意】、【异权之力】。

闪避反击·卡提希娅
成功闪避后一定时间内短按普攻，将会攻击目标，造成气动伤害。
```
