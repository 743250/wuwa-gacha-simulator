## 技能组

### 霜坠

```
普攻
进行最多4段的连续攻击，造成冷凝伤害。

重击
·当【霜色】达到上限时，长按普攻，将施放落霜坠。
·处于摇匣状态并攒满【霜色】时，松开普攻，将直接施放落霜坠。若釉瑚此时无法施放落霜坠，则直接进行一次问祯。
【霜色】的获取规则：
·当【霜色】未达到上限时，按住普攻，进入摇匣状态，持续积攒【霜色】。
·普攻命中敌人时，获得【霜色】。

重击·落霜坠
釉瑚将向前冲击，造成冷凝伤害，并进行一次问祯。

空中攻击
消耗耐力进行空中下落攻击，造成冷凝伤害。

闪避反击
成功闪避后，釉瑚未拥有祯祥的情况下，一定时间内短按普攻，将会向前冲击，造成冷凝伤害，并进行一次问祯。
```
