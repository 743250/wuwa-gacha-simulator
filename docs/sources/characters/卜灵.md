## 技能组

### 卦起雷动

```
普攻
进行最多4段的连续攻击，造成导电伤害。

重击·山雷颐
长按普攻，按从左到右顺序消耗高亮状态下的【卦象·艮】和【卦象·震】可施放重击·山雷颐，造成导电伤害。

重击·雷山小过
长按普攻，按从左到右顺序消耗高亮状态下的【卦象·震】和【卦象·艮】可施放重击·雷山小过，造成较少导电伤害，额外削减共振度。

重击·艮为山
长按普攻，按从左到右顺序消耗高亮状态下的【卦象·艮】和【卦象·艮】可施放重击·艮为山，回复附近队伍中所有角色的生命值。

重击·震为雷
长按普攻，按从左到右顺序消耗高亮状态下的【卦象·震】和【卦象·震】可施放重击·震为雷，持续回复附近队伍中所有角色的生命值，每秒回复1次，持续8秒。

重击·鬼门占卦
长按普攻时，若不足2枚【卦象】，则卜灵进入卜卦失败状态，消耗全部【卦象】并损失20%的当前生命值；当卜灵生命低于1%时，不消耗生命值。

空中攻击
消耗耐力进行空中下落攻击，造成导电伤害。

闪避反击
成功闪避后一定时间内短按普攻，将会施放普攻第3段攻击目标，造成导电伤害。
·施放闪避反击后一定时间内短按普攻，将会施放普攻第4段。

【卦象】获取规则
【卦象】上限4枚，达上限时，再次获取【卦象】，所有【卦象】将左移1格，最左侧【卦象】将被移除。
·普攻第2段造成伤害时获得【卦象·艮】。
·普攻第4段、空中攻击造成伤害时获得【卦象·震】。
·施放共鸣技能·遁影牵雷时获得【卦象·震】。
```
