## 技能组

### 一、二、三&hellip;&hellip;

```
普攻
进行最多4段的连续攻击，造成气动伤害。
施放普攻第4段后进入解读状态，持续5秒，若切换至其他角色则该效果提前结束。
若西格莉卡至少拥有50点【句点】，普攻从第2段开始。

普攻·明悟
处于解读状态时，地面短按普攻施放普攻·明悟，造成气动伤害，本次伤害为声骸技能伤害。
施放时退出解读状态。

重击
消耗耐力攻击目标，造成气动伤害。
施放重击后的一段时间内，短按普攻施放普攻第2段。

空中攻击
消耗耐力进行空中下落攻击，造成气动伤害。
施放空中下落攻击后的一段时间内，短按普攻施放普攻第2段。

闪避反击
在地面成功闪避后一定时间内短按普攻，将会攻击目标，造成气动伤害。
施放闪避反击后的一段时间内，短按普攻施放普攻第4段。

闪避反击·解读
处于解读状态时，在地面成功闪避后一定时间内短按普攻，将会施放闪避反击·解读，攻击目标，造成气动伤害，本次伤害为声骸技能伤害。
施放时退出解读状态。

空中闪避反击
在空中成功闪避后一定时间内短按普攻，将会攻击目标，造成气动伤害。
施放空中闪避反击后的一段时间内，短按普攻施放普攻第2段。
```
