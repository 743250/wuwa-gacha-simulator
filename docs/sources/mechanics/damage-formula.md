## 1. 属性与战斗公式

### 1.1 属性

鸣潮角色面板 16 种属性:攻击、防御、生命、攻击加成、防御加成、生命加成、暴击率、暴击伤害、共鸣效率(终极大招能量积攒速度)、元素伤害提升、元素伤害减免、治疗加成、普攻伤害提升、重击伤害提升、爆发技能伤害提升、终极大招伤害提升。

- **6 种元素**: 气动 / 导电 / 热熔 / 湮灭 / 冷凝 / 衍射。一个角色只能造成 1 种元素伤害。元素伤害减免最高 40%。
- **4 种攻击类型加成**: 普攻 / 重击 / 爆发技能 / 终极大招各独立配"伤害提升"。策划拆分乘区为投放数值留空间。

### 1.2 战斗公式(关键)

鸣潮用**乘法公式**(参考魔兽世界):

```
最终伤害 = 最终攻击 × (1 - 免伤率)
         × 技能自身伤害率
         × (1 + 指定技能伤害提升)
         × (1 + 延奏buff伤害提升)
         × (1 + 元素伤害提升 + 指定攻击类型伤害提升)
         × 暴击伤害
         × (1 - 元素伤害减免)
         × (1 - 战场元素伤害减免)
```

**最终攻击** = (角色攻击力 + 武器攻击力) × (1 + 攻击加成) + 装备(声骸)攻击力
**最终防御** = (角色防御力 + 武器防御力) × (1 + 防御加成) + 装备防御力
**最终生命** = (角色生命值 + 武器生命值) × (1 + 生命加成) + 装备生命力

### 免伤率(魔兽世界公式)

```
免伤率 = 防守方防御力 / (800 + 8 × 攻击方角色等级 + 防守方防御力)
```

把攻击方等级加进分母是**等级压制效果**:
- 攻击方等级越高，相同防御力下免伤率越低(打掉的伤害更多)
- 反过来，玩家到新地图面对更高级怪需要堆防御维持免伤率，**抑制越级打怪**

**这是模拟器 calcDamage 必须采用的公式**(原 `dmg = dmg - def×0.4` 减法是错的)。

### 技能自身伤害率

每个技能本身的倍率，例如重攻击 120%、普攻每段 30% 等。

### 指定技能伤害提升

只给指定名称的技能提升伤害。例:忌炎被动"枪扫风定 伤害提升20%";忌炎 6 链"苍躣八荒·后动伤害倍率提升120%"。

### 延奏 buff 伤害提升

切人时退场英雄给下一登场英雄的延奏 buff 是**独立乘算**(单独乘区)。例:维里奈 30 秒 15% 全元素伤害;莫特斐 14 秒 38% 重击伤害。

### 元素伤害提升

成长线描述中"气动伤害加成 / 热熔伤害加成"的统称。**只有对应元素角色才能吃到**。

### 指定攻击类型伤害提升

武器、声骸投放的 4 类加成。

### 暴击伤害

触发暴击时乘暴击伤害。

### 元素伤害减免

所有怪物**固定 10% 元素伤害减免**;特定副本额外 +30% 对指定元素的减免。

### 战场元素伤害减免

特定玩法(如无音区)怪物额外获得固定元素伤害减免。

## 2. 设计哲学拆解

### 拆乘区提升属性投放性价比

- **同括号加法**: 100 × (1+40%) = 140
- **拆括号乘算**: 100 × (1+20%) × (1+20%) = 144

鸣潮把每条养成线拆独立乘区:玩家每条成长线都有追求动力。

### 减伤也用乘法

不放 `(1+加成-减免)` 而放 `(1+加成) × (1-减免)`,**乘区拆开后减伤最终效果更强**。

### 乘区划分对应养成线

| 乘区 | 来源 | 玩家追求 |
|---|---|---|
| 指定技能伤害提升 | 角色被动 + 共鸣链(升星) | 升技能、抽链 |
| 延奏 buff | 切人退场技能 | 培养多角色 |
| 元素伤害提升 + 攻击类型提升 | 武器 + 声骸 | 抽武器刷装备 |
| 元素伤害减免 | 怪物自带(10% + 玩法 30%) | 多元素队伍 |

## 3. 战斗策略性

### 战中策略三分

**闪避 / 弹反 / 打断** + 能量收集 + 延奏 buff 堆叠。

### 切人 CD 1 秒

切人无 CD 等待，A 技能 CD 期间切 B 输出，再切回 A，循环积攒能量。**A 技能后摇切 B 时 A 残影留场把后段伤害打完**(残影无敌)。

### 延奏 buff 配合大招协同

经典连段:维里奈大招 → 延奏(15% 全元素，30秒) → 莫特斐入场大招 → 延奏(38% 重击，14秒) → 忌炎入场大招(开大招后所有攻击转重击)。

延奏 buff 独立乘算 + 2 个退场角色协同出击，最大化输出。

### 局内策略结论

研究技能 CD 与能量产出，做最佳出招表;闪避 BOSS 技能要求保留玩家操作空间。

## 4. 养成线

角色:升级 + 突破 / 升星(共鸣链) / 升技能(进阶)。
武器:升级 + 突破 / 升星(精炼)。
声骸(5 件):升级 + 解锁词条(4 副词条，不可洗练)。

道具混用注意:呓语声核同时用于武器突破/角色突破/技能升级。

## 5. 玩法

动作 ACT 为主，BOSS 约十余种通过不同数值强度复用，无音区/深塔等玩法均引用同一批 BOSS。

资源产出体力控制，氪金拉满后剩刷声骸。

## 6. 与本模拟器对照

模拟器是**AP 回合制简化版**,非官方复刻。下面是来自本文需引用回馈到代码的部分:

| 官方公式 | 模拟器需采用 |
|---|---|
| 免伤率 = DEF/(800+8×LV+DEF) | ✅ calcDamage 改为乘区 `× (1 - mitigation)` |
| 最终攻击 = (角色+武器) ×(1+加成)+装备 | ✅ 已用 `computeBattleStats` 聚合 |
| 元素伤害减免 10%+30%(玩法) | 模拟器简化为 `resist 0.10/0.40` |
| 延奏 buff 独立乘算 | ✅ 切人入场延奏 buff 通过 `variationBonus` 单乘区 |
| 4 类攻击类型独立乘区 | ✅ typeBonus + normalBonus/skillBonus/heavyBonus/burstBonus |
| 协奏值满 100 触发延奏 | ✅ concerto 满值 |
| 元素伤害减免限 40% 上限 | 设计文档已说明 |

鸣潮**动作 ACT 与本模拟器 AP 回合的根本差异**: 残影后摇切人、闪避、弹反不在模拟器抽象里。
