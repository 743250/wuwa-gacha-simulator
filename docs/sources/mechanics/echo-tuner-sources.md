## 1. 副本掉落

### 无音区（Tacet Field）
官方wiki标注所有4档调谐器均来自"无音清剿获得"。

模拟器实装数据（`src/battle/dungeon.js`）：
- **无音区·常规**（60波片）：`echo_tuner: 1`
- **无音区·高阶**（60波片）：`echo_tuner: 2`
- **角色专属无音区**（60波片，卡提希娅/珂莱塔/菲比等）：`echo_tuner: 1` 每个

官方明确数字：wiki 仅标明"无音清剿获得"，单次具体数量 wiki 未给出。

**未查到**：战歌重奏/世界BOSS在官方是否掉调谐器，社区文章未提及。

## 5. 档位比例

官方wiki对4档调谐器的描述一致，标注来源同为：
- **瑝珑探索获得**（大世界/地图资源）
- **无音清剿获得**（无音区副本）

**未查到具体比例。** 以下为推断：

- 初级调谐器：仅在低等级无音区或低世界等级时掉落，品质绿，用于二星声骸
- 中级调谐器：常规无音区常见掉落，品质蓝
- 高级调谐器：高阶无音区首要产出，品质紫
- 特级调谐器：高阶无音区/高等地图探索产出，品质金

**无官方确认的数据。** 模拟器当前仅用 `echo_tuner` 单位计数，未分4档。

## 6. 模拟器可用配置建议

以下数字为基于线索的推断，**非官方确认**：

| 渠道 | 单次量 | 频率 | 说明 |
|------|--------|------|------|
| 无音区·常规 (60波片) | 2 个（中级） | 不限，消耗波片 | 主要来源 |
| 无音区·高阶 (60波片) | 2-3 个（高级+少量特级） | 不限，消耗波片 | 主要来源 |
| 大世界探索（瑝珑） | 少量（初级/中级） | 一次性+每日刷新 | 低收益补充 |
| 每日委托 | 0 | 每日 | 社区未提及 |
| 商店 | 0 | - | 社区未提及 |
| 电台/深塔/海墟 | 0 | - | 社区未提及 |

**模拟器推荐**：保持当前 `echo_tuner` 单一计数（不分档），默认初始 5 个，无音区常规掉 1、高阶掉 2。如需分档，建议初级/中级绑定低等级无音区，高级/特级绑定高阶无音区。
