# 冥歌海墟（Whimpering Wastes）官方资料

> 采集日期：2026-06-25  
> 来源：Wuthering Waves Wiki / Game8 / Icy Veins / 多源交叉核验  
> 实装版本：v2.1 起永久玩法  
> 状态：**官方数据采集完成** —— 模拟器实现与官方差距见末尾对照表

---

## 1. 基本概况

冥歌海墟是 v2.1 引入的永久性挑战玩法，不消耗结晶波片。机制核心是**积分制 + 双队轮换 + 信物选择 + 焚潮状态**。

| 属性 | 官方设定 |
|---|---|
| 波片消耗 | 0（不消耗） |
| 重置周期 | 每月（每个版本一次） |
| 首次通关奖励 | 星声（一次性，每月首发 ~800） |
| 重复奖励 | 贝币 + 调谐器（仅第 12 层） |
| 核心机制 | 双队轮换 / 信物选择 / 焚潮 / 时间积分 |

## 2. 层级结构

### 2.1 三层结构（12 层）

| 区域 | 层数 | 类型 | 说明 |
|---|---|---|---|
| 浅层海域（Shallow Seas） | 1–6 | **一次性** | 教程关卡，稳定产出 |
| 深层海域（Deep Seas） | 7–12 | **周期性重置** | 高难层，每月轮换 |

**双队制**：每层分上半区 + 下半区，各用一支 3 人队伍（不同角色不可重复使用）。

### 2.2 当前模拟器实现

| 维度 | 官方 | 模拟器 |
|---|---|---|
| 层数 | 1-6（一次性）+ 7-12（周期性）| 5 关全部一次性（w1-w5）|
| 双队制 | 每层上下半区各 1 队 | 未实装（单队制） |
| 层选择 | 自由选择任意已解锁层 | 线性 1→2→3→4→5 |
| 重置 | 每月 1 日重置 7-12 层 | 每个版本重置全部 |

## 4. 信物系统（Token / 灵物）

### 4.1 官方机制

每层开战前从随机出现的信物中**选择 1 个**，同一种可重复选（效果叠加）。信物分三档：

| 档位 | 稀有度 | 示例效果 | 轮换规则 |
|---|---|---|---|
| 绿色 | 常见 | 基础数值加成 | 固定池，每月不换 |
| 紫色 | 稀有 | 中等机制加成 | 固定池，每月不换 |
| 金色 | 传说 | 改变玩法的核心增益 | **每月轮换**（4 个金色中随机选出现） |

**官方已知信物池（部分，30+ 种）**：

| 类型 | 信物名（中/英） | 效果方向 |
|---|---|---|
| 攻击类 | 强袭之印 | 全队攻击力提升 |
| 防御类 | 坚壁之印 | 全队 HP + 防御提升 |
| 速度类 | 迅捷之印 / Swift Strike | 动作速度提升 |
| 爆发类 | 焚潮之印 / Burning Tide | 加速焚烬积蓄 |
| 暴击类 | 会心之印 / Critical Edge | 暴击率 + 爆伤提升 |
| 治疗类 | 愈合之印 / Healing Pulse | 自动回复 |
| 蓄力类 | 蓄势之印 / Charged Power | 积累能量后爆发 |
| 协同类 | 协同之印 / Synergy | 切换角色时触发额外伤害 |
| 抵抗类 | 抗性之印 / Resilience | 元素抗性提升 |
| ... | 更多约 20+ 种 | — |

### 4.2 当前模拟器实现

模拟器仅实现 **6 种信物**（强袭/坚壁/迅捷/焚潮/会心/愈合），全为自定义效果，未与官方绿色/紫色/金色池对齐。

| 模拟器信物 | 效果 | 官方对应 |
|---|---|---|
| 强袭之印 | 全队攻击力 +25% | 大致对应官方绿色攻击信物 |
| 坚壁之印 | 全队 HP +30% · 防御 +20% | 大致对应官方绿色防御信物 |
| 迅捷之印 | 每回合 +1 AP | 无直接对应（回合制特有） |
| 焚潮之印 | 焚潮积蓄速度 +50% | 大致对应官方焚潮信物 |
| 会心之印 | 暴击率 +15% · 暴击伤害 +20% | 大致对应官方紫色暴击信物 |
| 愈合之印 | 每回合结束恢复 8% HP | 大致对应官方治疗信物 |

## 5. 焚潮机制（官方 vs 模拟器）

### 5.1 官方机制：焚烬（Burning Embers）

海墟的独特战斗资源，通过**快速造成伤害**积累。核心特点：

| 参数 | 官方设定 | 模拟器设定 |
|---|---|---|
| 积蓄方式 | 造成伤害积累焚烬值 | 回合计数（每 3 回合触发） |
| 触发条件 | 焚烬值满自动触发 | 回合条件触发 |
| 效果 | 全场 +60% 伤害，持续 30s | 全场 +30% 伤害，持续 2 回合 |
| 可叠加层数 | 触发后永久 +30% 伤害（常驻） | 仅一次性 buff |

### 5.2 焚潮信物（Burn Tide Token）
- 官方：全场持续火雨伤害，可辅助焚烬积蓄
- 模拟器：简化为焚烬积蓄速度 +50%

## 6. 评分系统

### 6.1 官方：时间积分制

官方采用**纯时间积分**——击杀越快，分数越高。每层有独立的积分目标：

| 档位 | 积分要求 | 星评 |
|---|---|---|
| 基础目标 | ~3,200/层 | ★ |
| 进阶目标 | ~4,000/层 | ★★ |
| 满分目标 | > 层满分数 | ★★★ |

关键：
- 积分完全取决于**清理速度**，不涉及血量/回合
- 允许重复挑战刷分，取最高分
- 12 层累计积分决定最终奖励档位

### 6.2 模拟器：回合+血量积分制

模拟器使用自创的「基础分 + 回合奖励 + 血量奖励」算法，与官方纯时间积分差异较大。

| 参数 | 官方 | 模拟器 |
|---|---|---|
| 积分依据 | 清怪速度（时间） | 通关回合 + 剩余血量 |
| 分数范围 | 无硬上限（取决于击杀速度） | 基础分 + 最多 500（回合）+ 最多 300（血量） |
| 五星条件 | 无（积分制非星评制） | ★1/★2/★3（复用深塔评星） |

## 9. 后续对齐建议

### 优先级高（影响核心体验）

1. **双队制**：海墟的核心特色是"需要 2 支不同队伍"——这与深塔的"3 塔需要 3 队"形成差异化逼氪压力。建议至少实现上下半区各 1 队

2. **焚潮机制**：从"回合计数"改为"伤害积蓄"驱动（更快杀敌→更快攒满→更快清场）。这是海墟区别于深塔的关键手感

### 优先级中（丰富策略深度）

3. **信物池扩展**：从 6 种扩展到 ~12 种，加金色传说级信物（如蓄势之印、协同之印），并做每月轮换金色信物

4. **时间积分**：考虑把积分从"回合"改为"剩余 AP/时间"相关

### 优先级低（锦上添花）

5. **12 层结构**：在保持模拟器简洁性的前提下，分 1-6 层（一次性）和 7-12 层（周期性）
6. **首月奖励**：首月通关 1-12 层应发放额外 800 星声
