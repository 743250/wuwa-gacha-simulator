# 官方资料索引

本目录记录从官方/半官方来源采集的原始设计资料，用来支撑角色、武器、敌人和机制移植。

## 资料优先级

1. **库街区官方共建 Wiki API**：优先记录技能组、共鸣链、武器文案、敌人机制原文。
2. **游戏内文本 / 官方公告**：用于核验 Wiki 缺失或更新滞后的信息。
3. **玩家强度榜 / 社区评价**：只用于判断数值天花板与玩法记忆点，不替代官方技能文本。
